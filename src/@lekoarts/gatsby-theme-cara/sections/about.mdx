# About Me

私は主にサーバー側を主体とするエンジニアです。

私のバックボーンは文学部史学科と異色ですが、プログラミングの面白さにハマり独学から現在の職にまで辿り着きました。

エンジニアとしての信念として、「愛着の持てるものづくり」というのを大切にしています。

誰かの役に立つという確信と、そのために注ぎ込んだアイデアや工夫が、実際に誰かの価値となって実を結ぶ―そんな深い結びつきのあるものを作り続けたいと考えています。

大切にしている価値観が2つあります。
1つ目は、「創意工夫を重ねた価値の創造」です。

単なる機能の実装ではなく、ユーザーにとって本当に価値のあるものを作るため、常に新しいアイデアを模索しています。技術的な制約があっても、その中で最大限の工夫を凝らすことで、より良い解決策を見出すことを心がけています。

2つ目は、「誠実なコミュニケーション」です。

チーム開発の成功には、メンバー間の信頼関係が不可欠です。、技術的な議論であっても、相手の立場に立った分かりやすい説明を心がけています。誠実なコミュニケーションを通じて、チームの一体感を高め、開発の効率化を実現することを重視しています。

これからも「誰かのために」という想いを大切に、技術とアイデアを組み合わせた価値あるソリューションを創造し続けていきたいと考えています。使う人の心に残る、愛着の持てるサービスやプロダクトを作り続けることが、エンジニアとしての私の目標です。
